# 🎵 מוזיקה חרדית

אגרגטור מוזיקה חרדית עם תמלול אוטומטי.

## תכונות
- סריקת שירים חדשים מ-YouTube (100+ אמנים חסידיים)
- תמלול אוטומטי של מילות שירים דרך Gemini AI
- תרגום שירים באנגלית לעברית
- ממשק תצוגה מודרני עם חיפוש וסינון

## הגדרת Secrets
בהגדרות הריפו → Settings → Secrets → Actions:
- `YT_API_KEY` – מפתח YouTube Data API
- `GEMINI_KEY` – מפתח Gemini API

## ארכיטקטורה
- GitHub Actions רץ כל 4 שעות
- סורק YouTube לפי רוטציה של אמנים (8 בכל ריצה)
- מתמלל עד 5 שירים חדשים בכל ריצה
- Frontend סטטי ב-GitHub Pages
